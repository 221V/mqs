MQS: Message Queue Client Library
=================================

RabbitMQ client library, routing keys, RPC over MQ and other stuff.

    [subscribe,user,User,group,Group]
  
turns to MQ subscription key

    <<"user.subscribe.maxim.group.synrc">>
